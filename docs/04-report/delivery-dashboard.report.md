## 6. 기술적 해결 사항

### re.subn JSON 개행 버그 (Critical)
- **문제**: `re.subn(replacement_string, html)` 시 교체 문자열의 `\n`이 실제 개행으로 해석 → 스타일 히스토리 컬럼 데이터에 raw newline 삽입 → `json.loads` 실패
- **해결**: `pattern.subn(lambda m: replacement, html)` — 함수형 교체 시 이스케이프 해석 없음
- **적용**: `replace_between()` 함수 내 모든 교체 구문에 적용

### Excel 날짜 시리얼 변환
- Excel 날짜 직렬 값(예: 46000) → `datetime(1899, 12, 30) + timedelta(days=N)` 방식으로 `YYYY-MM-DD` 문자열 변환
- `to_date_str()` 유틸리티 함수 추가
