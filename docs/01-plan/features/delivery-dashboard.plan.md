# Plan: 발주입고현황 비교 대시보드 웹앱

> **Feature ID**: delivery-dashboard
> **작성일**: 2026-03-12
> **담당자**: 통합소싱팀
> **보고 대상**: CEO, 기획팀, 영업팀
> **PDCA Phase**: Plan ✅

---

## 1. 배경 및 목적

### 비즈니스 컨텍스트
- DUVETICA 통합소싱팀은 26SS 시즌 입고율이 전년(25SS) 대비 낮아진 원인을 설명해야 함
- CEO 및 유관부서(기획, 영업)와 입고현황 회의 필요
- **핵심 메시지**: SPOT 오더 비중 급증(13.7% → 28.9%)이 전체 입고율 하락의 주요 원인

### 문제 정의
| 지표 | 25SS | 26SS | 변화 |
|------|------|------|------|
| 전체 입고율 | 57.4% | 52.5% | ▼ 4.9%p |
| SPOT 오더 비중 | 13.7% | 28.9% | ▲ 15.2%p |
| SPOT 입고율 | 41.6% | 32.8% | ▼ 8.8%p |
| MAIN 입고율 | 73.2% | 61.1% | ▼ 12.1%p |

---

## 2. 목표 (Goals)

### Primary Goal
- CEO 보고용 **한눈에 보이는 인사이트 대시보드** 제작
- SPOT 오더가 입고율에 미치는 영향을 데이터로 시각화

### Secondary Goals
- 25SS vs 26SS 동기간 비교 분석
- 복종별(top/bottom/outer/down/acc) 발주·입고 현황 파악
- 발주월별 입고 진도율 추이 비교
- 통합소싱팀 회의 자료로 활용

---

## 3. 주요 기능 요구사항

### F-01: KPI 요약 카드 (최상단)
- 25SS vs 26SS 전체 입고율 비교 (대형 숫자 표시)
- 발주 스타일수 / 발주수량 / 입고수량 / 입고금액 비교
- 전년 대비 변화율 표시 (▲▼)

### F-02: SPOT 오더 영향 분석 섹션 (CEO 핵심 메시지)
- SPOT 비중 변화 시각화 (Pie/Donut 차트)
- 오더구분별(MAIN/SPOT/RE-ORDER) 입고율 비교 막대그래프
- "SPOT이 없었다면 입고율은 XX%였을 것" 가상 시나리오 표시

### F-03: 복종별 비교표 (구분별)
- top/bottom/outer/down/acc 별 25SS vs 26SS 비교
- 발주수량, 입고수량, 진도율, 스타일수, 입고금액
- 진도율 색상 코딩 (70%↑ 초록, 50~70% 노랑, 50%↓ 빨강)

### F-04: 발주월별 입고 진도율 (동기간 비교 그래프)
- 7월~3월 발주월 기준 입고 진도율 꺾은선 그래프
- 25SS(파란선) vs 26SS(주황선) 나란히 비교
- 동기간 비교를 통해 시즌 진행 속도 파악

### F-05: 복종(women/men) 별 비교
- Women vs Men 발주·입고 현황
- 시즌별 구성비 변화

### F-06: 발주월별 상세 데이터 테이블
- 월별 발주수량, 입고수량, 진도율 표
- 25SS/26SS 나란히 비교

---

## 4. 비기능 요구사항

### 기술 스택
- **단일 HTML 파일** (서버 불필요, 바로 브라우저에서 열기)
- **Chart.js** (CDN 사용, 차트 라이브러리)
- **Vanilla JS + CSS** (프레임워크 없음, 배포 간편성 우선)
- 데이터는 JavaScript 변수로 하드코딩

### 디자인 요구사항
- CEO 보고용: 깔끔하고 전문적인 다크 테마
- DUVETICA 브랜드 컬러 반영 (고급스러운 감성)
- 인쇄 가능한 레이아웃 고려
- 반응형 (노트북 화면 기준, 1440px)

### 성능
- 단일 파일, 외부 CDN만 사용
- 오프라인 환경에서도 동작 (CDN 실패 시 기본 표시)

## 5. 핵심 인사이트 (스토리라인)

CEO 보고 시 전달할 스토리:

> **"26SS 전체 입고율 52.5%는 25SS(57.4%)보다 낮지만,
> 이는 SPOT 오더 비중이 13.7% → 28.9%로 급증했기 때문입니다.
> MAIN 오더만 보면 61.1%로, SPOT이 없었다면 훨씬 높은 진도율을 보였을 것입니다.
> SPOT 오더는 특성상 납기가 짧아 입고 진도율(32.8%)이 낮을 수밖에 없으며,
> 이는 시즌 후반에 집중적으로 입고될 예정입니다."**

## 7. 리스크 및 고려사항

| 리스크 | 대응방안 |
|--------|---------|
| RE-ORDER 발주금액이 formula(=L*Q)로 저장되어 파싱 불가 | 납품가×발주수량으로 계산 처리 |
| 입고금액이 일부 0으로 표시 | 실제 입고 완료분만 집계, 별도 안내 |
| SPOT의 낮은 진도율이 부정적으로 해석될 수 있음 | SPOT의 특성(단납기) 설명 텍스트 추가 |
